# Project 1: Calculator

## Description
A feature-rich command-line calculator with history support.

## Features
- Basic arithmetic (+, -, *, /)
- Power and modulo operations
- Square root
- Calculation history (last 10 operations)
- Input validation

## How to Run
```bash
python calculator.py
```

## Skills Used
- Functions
- Loops (while)
- Conditionals (if/elif/else)
- Error handling (try/except)
- Dictionaries
- String formatting (f-strings)
